# Raken

Enhancement for `rake`

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'raken'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install raken

## Usage

`raken --help`, you can see 3 more options(`--time [all]`, `--trace-tree OPTS`, `--pry [csnfe]`) prepended.
